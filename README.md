# Event-Triggered-Fixed-Time-Optimal-Tracking-Control
Event-Triggered Fixed-Time Optimal Tracking Control for Stochastic Pure-Feedback Systems with Input Delay via Reinforcement Learning

These materials are sufficient to reproduce all figures and quantitative results presented in this paper.
	
The complete source code of the core control algorithm is not publicly disclosed, as it is part of an ongoing proprietary industrial project at the Xi'an Microelectronics Technology Institute and involves intellectual property subject to confidentiality obligations. Nevertheless, it can be obtained from the corresponding author upon reasonable request, subject to institutional approval and, where applicable, the execution of a material transfer agreement for non-commercial academic use.
